# 06_FRONTEND_REACT

This directory contains all the frontend engineering classes, specifically focusing on the React.js ecosystem. 
This was separated from the backend engineering modules to maintain a clean and logical separation of concerns (Frontend UI vs Backend APIs).

## Directory Structure

Each class session follows the standardized layout:

```text
class-name/
|
+-- class-notes/    -> handwritten or markdown notes from class
+-- class-code/     -> instructor reference code or cohort code
+-- practice/       -> my personal code practices for the topic
+-- assignments/    -> official cohort assignments
+-- README.md       -> high level summary of the class
+-- links.md        -> resource links and tracking status
```
