# HTML Resume Page Assignment

This README explains the HTML-only approach used in `index.html` for the cohort assignment.

## Approach

- Built the page using only HTML, as required by the assignment.
- Used semantic tags to separate content into sections for readability.
- Added `<header>` for name and role, `<main>` for content, and `<footer>` for closing details.
- Used `<section>` for each resume block: About, Skills, Experience, Projects, Education, and Contact.
- Used `<ul>` for skills and `<table>` for experience and education to keep the layout structured.
- Used `<article>` for each project entry to show project details clearly.
- Added `<hr>` elements to separate sections visually without CSS.

## Assignment structure in `index.html`

- Header / name section
- About section with a personal summary
- Skills list using a clean unordered list
- Experience section with a table layout and three rows
- Projects section describing two real example projects
- Education section with a table layout and institution details
- Contact section with email, LinkedIn, portfolio, GitHub, and location

## Why this satisfies the assignment

- It follows the cohort instructions for an HTML-only resume page.
- It keeps the structure clean and easy to read.
- It uses proper HTML tags and semantic organization.
- It shows practical learning by using HTML elements correctly, not styling.

## How to view

1. Open `index.html` directly in a browser.
2. The page displays the resume structure without CSS.
