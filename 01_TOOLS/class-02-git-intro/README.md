# Tools - Orientation Class, Introduction to Git

## Overview

This class introduced the concept of version control and why it is essential in modern development.

The session focused on understanding real-world problems developers faced before Git and how version control systems solve them.

---

## What I Learned

### Why Version Control Exists

- Problems like losing code and managing multiple versions
- “Pendrive problem” and lack of tracking
- Difficulty in collaboration before version control

---

### Introduction to Git

- What is Git
- Why Git is widely used in development
- Basic understanding of repositories

---

### Developer Workflow Context

- Importance of version control in teams
- Tracking changes and maintaining history
- Foundation for future Git commands and workflows
