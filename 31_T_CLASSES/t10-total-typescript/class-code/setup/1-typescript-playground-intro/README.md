# Setup 1 - TypeScript Playground Intro

## Purpose

This is the lightest starting point for the T10 flow.

Before setting up TypeScript locally with npm, Vite, or Deno, this step shows the basic TypeScript idea in the simplest way:

- write a `.ts` file
- add type annotations
- see type checking instantly
- understand how TypeScript improves editor feedback before runtime

---

## Best Place To Try This

- TypeScript Playground
- VS Code with TypeScript support enabled

No package installation is required for this step.

---

## File In This Folder

- `basic-demo.ts` -> small intro example with variables, object typing, and a typed function

---

## Why This Comes Before Setup 2

`2-npm-local` is about local project setup.

This folder exists one step before that, so the learning sequence stays natural:

1. understand basic TypeScript syntax
2. run TypeScript locally with npm
3. use TypeScript with Vite
4. use TypeScript with Deno
