# T7 - DOM Mini Projects

## Overview

In this class, I learned the fundamentals of the DOM (Document Object Model) and how JavaScript interacts with HTML elements in the browser.
The session focused on real-time DOM manipulation and building interactive UI components.

---

## What I Learned

### DOM Basics

- What is DOM and how browser represents HTML as a tree
- Accessing elements using:
  - getElementById
  - querySelector / querySelectorAll
- Difference between NodeList and HTMLCollection

---

### DOM Manipulation

- Changing content (innerText, innerHTML)
- Updating styles dynamically
- Creating and removing elements
- Handling attributes

---

### Event Handling

- addEventListener
- Click, input, and change events
- Event-driven programming

---

## Practice Highlights

### Completed Projects

- Dynamic List Creator
- Color Palette Generator

### Pending Projects

## Key Takeaways
