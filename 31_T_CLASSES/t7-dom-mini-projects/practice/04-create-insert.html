✏️ Tasks

TASK 1

createElement — Basic Flow

  1. Ek naya <p> element banao
  2. Uska textContent "Mera naya paragraph!" rakho
  3. Usmein class "item-card" add karo
  4. Sandbox mein append karo (end mein add)
  5. Phir ek aur paragraph banao aur prepend karo (start mein)
📄 JavaScriptTASK 1
// ── TASK 1: createElement + append ──────────────────
const sandbox = document.getElementById('sandbox');

// Step 1: Element banao (memory mein hai, DOM mein nahi abhi)
const newPara = document./* createElement */('p');

// Step 2: Properties set karo
newPara.textContent = 'Mera naya paragraph!';
newPara.classList.add('item-card');

// Step 3: DOM mein daalo — END mein
sandbox./* append */(newPara);

// ── Prepend — START mein add ─────────────────────────
const firstPara = document.createElement('p');
firstPara.textContent = 'Main sabse pehle hoon!';
firstPara.classList.add('item-card');
firstPara.style.borderColor = 'rgba(61,220,151,0.4)';
sandbox./* prepend */(firstPara);
document.createElement('p') — element banao
parent.append(newEl) — END mein add
parent.prepend(newEl) — START mein add
Note: append strings bhi accept karta hai! parent.append('Hello text')
TASK 2

Remove karna — 2 tarike

  1. Sandbox ka last child remove karo — modern way se
  2. Sandbox ka first element child remove karo — old way (removeChild) se bhi dekho
📄 JavaScriptTASK 2
// ── TASK 2: Remove ──────────────────────────────────
const sandbox = document.getElementById('sandbox');

// Modern way — seedha element pe .remove() call karo
const lastChild = sandbox.lastElementChild;
if (lastChild) lastChild./* remove() */;

// Old way — parent.removeChild(child)
const firstChild = sandbox.firstElementChild;
if (firstChild) sandbox./* removeChild(firstChild) */;
TASK 3

Dynamic List Builder — Real Use!

Input mein text type karo, button dabaao — list mein item add ho. Ye real-world pattern hai!

  • Pehle se ek item hai
📄 JavaScriptTASK 3 — List Builder 
// ── TASK 3: Dynamic List ────────────────────────────
const addBtn   = document.getElementById('add-item-btn');
const listInput = document.getElementById('list-input');
const list      = document.getElementById('demo-list');

addBtn.addEventListener('click', function() {
  // 1. Input ka value lo
  const text = listInput.value.trim(); // trim() = extra spaces hatao

  // 2. Agar empty hai toh kuch mat karo
  if (text === '') return;

  // 3. Naya <li> banao
  const newItem = document./* createElement('li') */;
  newItem.textContent = text;

  // 4. List mein add karo
  list./* append(newItem) */;

  // 5. Input clear karo
  listInput.value = '';
  listInput.focus(); // cursor wapas input mein
});
input.value — input ka current text
text.trim() — start/end ke spaces hatao
document.createElement('li') — li element banao
list.append(newItem) — add to end
input.value = '' — input clear karo
CHALLENGE 🔥

Delete Button Wali List

Upar wali list ko upgrade karo — har naye item ke saath ek ✕ delete button bhi add ho. Delete button click karne se woh item remove ho jaaye.

📄 JavaScriptCHALLENGE
// ── CHALLENGE: Delete Button ─────────────────────────
// Pehle wale addEventListener ke andar — newItem banane ke baad:

// 1. Delete button banao
const delBtn = document.createElement('button');
delBtn.textContent = '✕';
delBtn.className = 'remove-btn';

// 2. Delete button par click listener lagao
delBtn.addEventListener('click', function() {
  newItem./* remove() */;  // parent ki zarurat nahi — seedha remove!
});

// 3. Button ko li ke andar add karo
newItem./* append(delBtn) */;

// 4. List mein add karo (pehle jaisa)
list.append(newItem);
