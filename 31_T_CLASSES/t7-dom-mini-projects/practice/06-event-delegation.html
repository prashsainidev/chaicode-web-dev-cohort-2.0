TASK 1

Basic Event Delegation Samjho

good-list pe ek listener hai jo saare li items handle karta hai. Neeche code mein dekho — e.target kaisa kaam karta hai.

📄 JavaScript — Delegation ka Core PatternPadho + Samjho 
// ── CORE PATTERN — Event Delegation ─────────────────

const goodList = document.getElementById('good-list');

// SIRF YEK LISTENER — ul (parent) pe
goodList.addEventListener('click', function(e) {

  // e.target = exactly jo click hua
  // Agar <li> click hua toh e.target = li element
  // Agar koi aur cheez click hui toh e.target = woh element

  if (e.target.tagName === 'LI') {    // sirf li clicks handle karo
    console.log('Li click hua:', e.target.textContent);
    e.target.classList.add('selected');
  }

  // YAAD RAKHO:
  // e.target    = jo actually click hua (li)
  // e.currentTarget = jahan listener hai (ul)
  console.log('target:',        e.target.tagName);
  console.log('currentTarget:', e.currentTarget.tagName);
});

Dynamic Items — Delegation Ki Asli Taakat

  1. good-list mein "Add Item" button pe listener lagao — naya li add ho

const goodList  = document.getElementById('good-list');
const addGoodBtn = document.getElementById('add-good');
let itemCount = 4;

// 1. New item add karo — koi naya click listener NAHI lagana
addGoodBtn.addEventListener('click', function() {
  const newItem = document./* createElement('li') */;
  newItem.textContent = 'New Item ' + itemCount++;
  goodList./* append(newItem) */;
  // Note: koi addEventListener nahi lagaya naye item pe!
  // Parent ka listener automatically handle karega 🎉
});

// 2. Parent listener (pehle se hai) — naye items bhi catch karega
goodList.addEventListener('click', function(e) {
  if (e.target.tagName === 'LI') {
    console.log('Clicked:', e.target.textContent);
    // Pehle selected remove karo sabse
    goodList.querySelectorAll('li').forEach(li => li.classList.remove('selected'));
    // Phir clicked item highlight karo
    e.target.classList.add('selected');
  }
});
Delegation ka key: listener sirf parent pe lagao.
e.target.tagName === 'LI' — check karo ki actually li click hua
Naye element add karne ke baad koi naya listener nahi chahiye — parent already sun raha hai!
e.target har baar naya element bhi pakdega.
TASK 3

Real Pattern — closest() se Safe Delegation

Jab li ke andar aur elements ho, e.target inner element ho sakta hai. closest() use karo safe delegation ke liye.

📄 JavaScriptTASK 3 — closest() Pattern 
// ── TASK 3: closest() — Safe Delegation ─────────────
const advList = document.getElementById('advanced-list');
const output  = document.getElementById('advanced-output');

advList.addEventListener('click', function(e) {

  // Problem: agar <strong> ya <span> click kiya toh
  // e.target = strong/span hoga, li nahi!

  // Solution: closest('li') — click se upar jaata hai
  // jab tak li nahi milta
  const li = e.target.closest('li');

  if (!li) return; // li nahi mila = ul ke bahar click hua

  // data-id attribute se item ka id lo
  const id = li.dataset.id;   // data-id="1" → dataset.id = "1"

  output.textContent = 'Clicked li with data-id: ' + id;
  output.style.color = '#4ecdc4';
});

Multi-Action Delegation

Ek list hai jisme har item pe Edit aur Delete — dono buttons hain. Sirf ek parent listener se dono handle karo — data-action attribute se pata karo kaunsa button click hua.

    // ── CHALLENGE: Multi-Action Delegation ───────────────
const actionList = document.getElementById('action-list');
const actionOut  = document.getElementById('action-output');

// List setup — 3 items banao
['Aarav', 'Priya', 'Rajan'].forEach(function(name, i) {
  const li = document.createElement('li');
  li.dataset.id = i + 1;
  li.innerHTML = `
    <span style="flex:1">${name}</span>
    <button data-action="edit"   style="background:rgba(124,106,247,0.15);border:1px solid rgba(124,106,247,0.3);color:#7c6af7;padding:4px 12px;border-radius:6px;cursor:pointer;font-size:12px;margin-right:6px;">Edit</button>
    <button data-action="delete" style="background:rgba(255,107,107,0.1);border:1px solid rgba(255,107,107,0.25);color:#ff6b6b;padding:4px 12px;border-radius:6px;cursor:pointer;font-size:12px;">Delete</button>
  `;
  li.style.cssText = 'display:flex;align-items:center;background:#13151a;border:1px solid rgba(255,255,255,0.08);border-radius:8px;padding:10px 16px;margin-bottom:8px;font-size:14px;';
  actionList.append(li);
});

// EK LISTENER — parent pe
actionList.addEventListener('click', function(e) {
  const btn = e.target.closest('button');
  if (!btn) return;

  const action = btn.dataset.action; // 'edit' ya 'delete'
  const li     = btn.closest('li');
  const id     = li.dataset.id;
  const name   = li.querySelector('span').textContent;

  if (action === 'edit') {
    actionOut.textContent = `✏️ Edit: ${name} (id: ${id})`;
    actionOut.style.color = '#7c6af7';
  } else if (action === /* 'delete' */) {
    li.remove();
    actionOut.textContent = `🗑️ Deleted: ${name}`;
    actionOut.style.color = '#ff6b6b';
  }
});
