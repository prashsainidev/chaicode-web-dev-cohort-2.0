TASK 2

Children Navigate Karo

  1. parent-el ke saare children print karo — children use karo
  2. parent-el ka pehla child aur aakhri child print karo
  3. Kitne children hain? children.length
📄 JavaScriptTASK 2
// ── TASK 2: Children Navigation ──────────────────────
const parent = document.getElementById('parent-el');

// 1. Saare children (sirf element nodes — text/whitespace nahi)
console.log('Children:', parent.children); // HTMLCollection

// 2. First aur Last child
console.log('First:', parent.firstElementChild);
console.log('Last:',  parent.lastElementChild);

// 3. Count
console.log('Total children:', parent.children.length);

// 4. Sab children pe loop karo
Array.from(parent.children).forEach(function(child) {
  console.log('Child id:', child.id);
});
TASK 3

Siblings Navigate Karo

  1. child-2 ka next sibling kya hai?
  2. child-2 ka previous sibling kya hai?
  3. child-1 ka next sibling — phir next sibling — chain karo
📄 JavaScriptTASK 3
// ── TASK 3: Siblings ─────────────────────────────────
const child2 = document.getElementById('child-2');

// Next sibling (Element wala — text nodes skip)
const nextEl = child2.nextElementSibling;
console.log('Next sibling:', nextEl.id);  // child-3

// Previous sibling
const prevEl = child2.previousElementSibling;
console.log('Prev sibling:', prevEl.id);  // child-1

// Chain — child1 se sab siblings traverse karo
const child1 = document.getElementById('child-1');
let current = child1;
while (current !== null) {
  console.log('Traversing:', current.id);
  current = current.nextElementSibling; // agle pe jao
}
// null milega jab koi sibling nahi bachta
TASK 4

closest() — Ancestor Search

closest() ek element se upar jaata hai aur pehla ancestor dhundh deta hai jo selector se match kare. Event delegation mein bahut kaam aata hai!

📄 JavaScriptTASK 4
// ── TASK 4: closest() ───────────────────────────────
const child2 = document.getElementById('child-2');

// Nearest ancestor jo .parent-box ho
const nearestParent = child2.closest('.parent-box');
console.log('Nearest .parent-box:', nearestParent.id); // parent-el

// Khud bhi match hota hai agar selector suit kare
const itselfCheck = child2.closest('.child-box');
console.log('closest self:', itselfCheck.id); // child-2 khud!

// Agar nahi milta toh null milta hai
const noMatch = child2.closest('.does-not-exist');
console.log('No match:', noMatch); // null
CHALLENGE 🔥

Traversal se Highlight Karo

child-2 click hone par — uske dono siblings ko highlight karo (border color change karo), lekin child-2 ko mat karo. Sirf traversal use karo — getElementById nahi!

📄 JavaScriptCHALLENGE
// ── CHALLENGE: Traversal + Highlight ────────────────
const child2 = document.getElementById('child-2');

child2.addEventListener('click', function() {
  // Get siblings via traversal — getElementById nahi use karna!
  const prev = child2.previousElementSibling;
  const next = child2.nextElementSibling;

  // Pehle reset karo
  child2.parentElement.querySelectorAll('.child-box').forEach(el => {
    el.style.outline = 'none';
  });

  // Siblings highlight karo
  if (prev) prev.style.outline = '2px solid #3ddc97';
  if (next) next.style.outline = '2px solid #3ddc97';

  console.log('Highlighted siblings of child-2!');
});
