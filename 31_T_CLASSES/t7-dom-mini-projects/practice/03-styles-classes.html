TASK 1

element.style — Directly Style Change Karo

  1. card1 ka background color #2d1b4e karo
  2. card1 ka color white karo
  3. card1 ka border radius 20px karo
  4. card2 ka font size 20px karo — CSS property name camelCase mein likho!
📄 JavaScriptTASK 1
// ── TASK 1: element.style ───────────────────────────
const card1 = document.getElementById('card1');
const card2 = document.getElementById('card2');

// 1. Background color
card1.style.backgroundColor = '#2d1b4e';   // background-color → camelCase!

// 2. Text color
card1.style./* color */ = 'white';

// 3. Border radius
card1.style./* borderRadius */ = '20px';  // border-radius → camelCase!

// 4. Font size (camelCase mein likhna)
card2.style./* fontSize */ = '20px';         // font-size → fontSize

// Verify: console mein dekho
console.log(card1.style.backgroundColor);
CSS properties camelCase mein likhte hain JS mein:
background-color → backgroundColor
border-radius → borderRadius
font-size → fontSize
margin-top → marginTop
TASK 2

classList — Best Practice Way

  1. card2 mein highlighted class add karo
  2. card3 mein danger class add karo
  3. Check karo — card1 mein "active" class hai ya nahi? (contains)
  4. card1 mein "active" class toggle karo — 2 baar run karo, kya hota hai dekho
📄 JavaScriptTASK 2
// ── TASK 2: classList ───────────────────────────────
const card1 = document.getElementById('card1');
const card2 = document.getElementById('card2');
const card3 = document.getElementById('card3');

// 1. Add class
card2.classList.add('highlighted');

// 2. Add danger class
card3.classList./* add */('danger');

// 3. Check karo — contains boolean deta hai
const hasActive = card1.classList./* contains */('active');
console.log('card1 mein active class hai?', hasActive); // false aana chahiye

// 4. Toggle — 1st call: add karta hai, 2nd call: remove karta hai
card1.classList./* toggle */('active');
console.log('After toggle:', card1.classList.contains('active')); // true
card1.classList./* toggle */('active');
console.log('After 2nd toggle:', card1.classList.contains('active')); // false
el.classList.add('class-name') — class add karo
el.classList.remove('class-name') — class hatao
el.classList.toggle('class-name') — agar hai toh hata do, nahi hai toh add karo
el.classList.contains('class-name') — true/false batata hai
TASK 3

Dark Mode Toggle Banana — Real Use Case!

Toggle Button pe click karo toh card1 ka background alternate karo — light aur dark ke beech. classList.toggle use karo.

📄 JavaScriptTASK 3
// ── TASK 3: Toggle Button ───────────────────────────
const toggleBtn = document.getElementById('toggle-btn');
const card1 = document.getElementById('card1');

toggleBtn.addEventListener('click', function() {
  // 1. card1 mein 'highlighted' class toggle karo
  card1.classList./* toggle */('highlighted');

  // 2. Button ka text bhi change karo based on current state
  if (card1.classList.contains('highlighted')) {
    toggleBtn.textContent = 'Normal Karo';
  } else {
    toggleBtn.textContent = 'Highlight Karo';
  }
});
