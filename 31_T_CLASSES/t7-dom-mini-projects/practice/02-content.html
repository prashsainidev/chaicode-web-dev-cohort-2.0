✏️ Tasks

TASK 1

textContent — Safe Text Change

  1. title-el ka text "Mera Naya Title" kar do
  2. desc-el ka text change karo — isme HTML tags likho aur dekho kya hota hai (textContent HTML ko parse nahi karta!)
  3. title-el ka current text console mein print karo
📄 JavaScriptTASK 1
// ── TASK 1: textContent ─────────────────────────────

// 1. title-el ka text change karo
const title = document.getElementById('title-el');
title./* property */ = 'Mera Naya Title';

// 2. desc-el mein HTML tags wala text set karo — dekho kya hoga
const desc = document.getElementById('desc-el');
desc.textContent = '<strong>Yeh bold nahi hoga!</strong> Tags as text show honge';

// 3. Current text read karo
console.log('Title ka text:', title./* textContent */);
element.textContent = 'naya text' — setter hai
element.textContent — getter hai (current value milti hai)
textContent HTML parse nahi karta — <strong> literally as text show hoga
TASK 2

innerHTML — HTML Parse Karta Hai

  1. desc-el mein actual bold text aur ek link insert karo — <strong> tag use karo
  2. Ek naya list <ul><li> insert karo title-el ke andar
  3. Console mein dekho: innerHTML aur textContent ka fark kya aata hai same element pe?
📄 JavaScriptTASK 2
// ── TASK 2: innerHTML ───────────────────────────────

// 1. innerHTML se actual bold text insert karo
const desc = document.getElementById('desc-el');
desc.innerHTML = '<strong>Yeh actually bold hoga!</strong> Aur yeh normal text.';

// 2. List insert karo
const title = document.getElementById('title-el');
title.innerHTML = /* '<ul><li>Item 1</li><li>Item 2</li></ul>' */;

// 3. Fark dekho console mein
console.log('innerHTML:', desc.innerHTML);    // HTML tags bhi dikhenge
console.log('textContent:', desc.textContent); // sirf text milega
TASK 3

Attributes — getAttribute / setAttribute

  1. Link ka href attribute read karo console mein
  2. Link ka href https://youtube.com kar do
  3. Image ka alt attribute change karo "Meri Image" kar do
  4. Image pe ek naya attribute add karo: data-id="42"
📄 JavaScriptTASK 3
// ── TASK 3: Attributes ──────────────────────────────

const link = document.getElementById('link-el');
const img  = document.getElementById('img-el');

// 1. href read karo
console.log('Current href:', link./* getAttribute('href') */);

// 2. href change karo
link./* setAttribute */('href', 'https://youtube.com');

// 3. alt change karo
img./* tumhara code */('alt', 'Meri Image');

// 4. Custom attribute add karo
img./* tumhara code */('data-id', '42');

// 5. Verify karo
console.log('New href:', link.getAttribute('href'));
console.log('data-id:',  img.getAttribute('data-id'));
Read: element.getAttribute('href')
Write: element.setAttribute('href', 'new-value')
Same method: setAttribute kisi bhi attribute ke liye kaam karta hai — built-in ya custom (data-*)
TASK 4

innerText vs textContent — Fark Pakdo

hidden-el ka style display:none hai. Dekho textContent uska text deta hai ya nahi — aur innerText deta hai ya nahi.

📄 JavaScriptTASK 4
// ── TASK 4: innerText vs textContent ────────────────
const hidden = document.getElementById('hidden-el');

// Dono se text read karo — kya fark aata hai?
console.log('textContent:', hidden.textContent);  // text milega? ya ''?
console.log('innerText:',   hidden.innerText);    // text milega? ya ''?

// Answer: textContent = text milega (CSS ignore karta hai)
//         innerText   = '' milega (display:none respect karta hai)
⚠️ XSS Warning — Kabhi Mat Karo:
element.innerHTML = userInput — agar userInput user ne type kiya ho toh ye dangerous hai. User kuch bhi inject kar sakta hai. User data ke liye hamesha textContent use karo.
CHALLENGE 🔥

Input se text lo aur page par show karo

  1. Neeche ek input aur button hai
  2. Button click hone par — input ka value lo aur #output-box mein safe tarike se dikhao
  3. Hint: textContent use karo — innerHTML mat karo
Output yahan aayega...
📄 JavaScriptCHALLENGE
// ── CHALLENGE: Input → Output ────────────────────────

const btn    = document.getElementById('show-btn');
const input  = document.getElementById('user-input');
const output = document.getElementById('output-box');

btn.addEventListener('click', function() {
  // 1. input ka current value lo
  const userText = input./* input ka value kaise milega? */;

  // 2. output box mein safely dikhao
  output./* textContent ya innerHTML? */ = userText;
});
Input ka value: input.value (textContent nahi, value property hai input elements mein)
Safe display: output.textContent = userText
