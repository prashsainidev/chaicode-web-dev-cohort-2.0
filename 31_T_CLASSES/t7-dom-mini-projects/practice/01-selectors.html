querySelector mein CSS selector ka use hota hai. Ek dot (.) class ke liye, hash (#) id ke liye.

📄 JavaScript

// 1. class "special" wala pehla element select karo
const specialEl = /* tumhara code */;

// 2. id "hero-box" select karo (# use karo)
const hero2 = /* tumhara code */;

// 3. input element select karo (tag name se)
const inp = /* tumhara code */;

// 4. nested: sandbox ke andar ka pehla .item
const firstItem = /* tumhara code: '#sandbox .item' */;

console.log(specialEl, hero2, inp, firstItem);
querySelector CSS selector leta hai:
Class: document.querySelector('.item')
ID: document.querySelector('#hero-box')
Tag: document.querySelector('input')
Nested: document.querySelector('#sandbox .item')

querySelectorAll se saare .item class wale elements select karo aur har ek ka textContent console mein print karo.

  console.log(/* item ka textContent */);

// 4. BONUS: allItems ko real Array mein convert karo
const itemsArray = /* Array.from(?) */;

Array conversion: Array.from(allItems)
CHALLENGE 🔥

Sab kuch khud likho

  1. Select karo sandbox ke andar sirf wo items jo "special" class bhi rakhte hain
