---

## What I Learned

### DDL And Constraints

- creating databases and tables
- using `PRIMARY KEY`, `UNIQUE`, `NOT NULL`, `DEFAULT`, and `CHECK`
- understanding how schema design protects data quality
- using `ALTER TABLE` when schema requirements change

---

### Querying And Data Changes

- selecting only the columns actually needed
- filtering with `WHERE`, `BETWEEN`, `IN`, `LIKE`, and `IS NULL`
- sorting with `ORDER BY`
- paginating with `LIMIT` and `OFFSET`
- using `INSERT`, `UPDATE`, and `DELETE` carefully with `WHERE`

---

### Aggregations

- using `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX`
- grouping records with `GROUP BY`
- filtering grouped results with `HAVING`
