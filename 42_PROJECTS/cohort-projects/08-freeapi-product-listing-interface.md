# FreeAPI: Product Listing Interface

**Cohort**: Web Dev Cohort 2026
**Timeline**: Start: 2 May 2026, 8:00 pm | Due: 6 May 2026, 11:59 pm

## Description
In this project, you will build a product listing interface using the Products API.

Your goal is to fetch product data from the API and display it in a structured format similar to a simple e-commerce layout.

**API Endpoint**:
`https://api.freeapi.app/api/v1/public/randomproducts`

Focus on designing a clean UI that presents product information clearly.

You may use CSS or Tailwind CSS for styling.
