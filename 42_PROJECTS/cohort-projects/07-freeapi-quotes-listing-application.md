# FreeAPI: Quotes Listing Application

**Cohort**: Web Dev Cohort 2026
**Timeline**: Start: 2 May 2026, 8:00 pm | Due: 6 May 2026, 11:59 pm

## Description
In this project, you will build a quotes listing interface using the Quotes API.

Your application should fetch quotes from the API and present them in a clean layout suitable for reading or browsing quotes.

**API Endpoint**:
`https://api.freeapi.app/api/v1/public/quotes`

You may design this as a quote gallery or a quote generator-style interface.

You may use CSS or Tailwind CSS for styling.
