# FreeAPI: Meals Listing Interface

**Cohort**: Web Dev Cohort 2026
**Timeline**: Start: 2 May 2026, 8:00 pm | Due: 6 May 2026, 11:59 pm

## Description
In this project, you will build a meals or recipes listing interface using the Meals API.

Your application should display meal information in a structured and readable layout.

**API Endpoint**:
`https://api.freeapi.app/api/v1/public/meals`

Design the interface in a way that makes browsing meals simple and visually clear.

You may use CSS or Tailwind CSS for styling.
