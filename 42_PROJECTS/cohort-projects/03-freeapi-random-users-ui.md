# FreeAPI: Random Users UI

**Cohort**: Web Dev Cohort 2026
**Timeline**: Start: 2 May 2026, 8:00 pm | Due: 6 May 2026, 11:59 pm

## Description
In this project, you will build a React-based user interface using the Random Users API.

Your goal is to read the API documentation, understand the structure of the response, and design a clean user interface that displays user information properly.

**API Endpoint**:
`https://api.freeapi.app/api/v1/public/randomusers`

Your application should fetch user data from the API and present it in a structured layout such as user cards or profile-style views.

You may use CSS or Tailwind CSS for styling.
