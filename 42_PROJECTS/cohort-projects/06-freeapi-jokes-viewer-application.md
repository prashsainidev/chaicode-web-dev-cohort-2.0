# FreeAPI: Jokes Viewer Application

**Cohort**: Web Dev Cohort 2026
**Timeline**: Start: 2 May 2026, 8:00 pm | Due: 6 May 2026, 11:59 pm

## Description
In this project, you will build a jokes viewer interface using the Random Jokes API.

Your application should fetch jokes from the API and display them in a readable and interactive format.

**API Endpoint**:
`https://api.freeapi.app/api/v1/public/randomjokes`

Design the UI in a way that makes browsing jokes simple and engaging.

You may use CSS or Tailwind CSS for styling.
