# FreeAPI: Random Cat Viewer

**Cohort**: Web Dev Cohort 2026
**Timeline**: Start: 2 May 2026, 8:00 pm | Due: 6 May 2026, 11:59 pm

## Description
In this project, you will build a random cat viewer application using the Random Cat API.

Your application should fetch and display cat images dynamically from the API.

**API Endpoint**:
`https://api.freeapi.app/api/v1/public/cats/cat/random`

Design a simple interface that updates the displayed image when new data is fetched.

You may use CSS or Tailwind CSS for styling.
