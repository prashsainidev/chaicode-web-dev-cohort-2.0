# Cohort Progress Tracker

This file tracks my cohort progress in a clearer way so I can see:

- which classes I attended live
- which classes still need rewatch
- what is documented
- what is practiced
- how blog writing is growing alongside the classes

Last updated: May 4, 2026

---

## Status Legend

### Class Tracking

- `Done` = completed
- `In progress` = started but not finished
- `Pending` = not started yet
- `Not applicable` = this column does not apply to that class
- `Upcoming` = class has not happened yet

### How To Read The Columns

- `Live` = whether I attended that class live
- `Date` = scheduled class date from the cohort dashboard
- `Rewatch` = whether I revised, rewatched, or covered the recorded class after missing it live
- `Notes` = whether my repo notes / README work is done
- `Practice` = whether actual code practice is done
- `Assignment` = whether assignment work is done
- `Blog` = related blog coverage status or count

Note:

- this repo is intentionally pre-structured
- some folders exist before the actual work starts
- `Blog` can show a count like `3 published` when one class or topic expands into multiple blog posts
