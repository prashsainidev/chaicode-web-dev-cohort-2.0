# 06_BUILDING_MODERN_FRONTENDS

This directory contains the frontend engineering classes focused on React, component thinking, and modern UI application flow.

It is separate from backend engineering so frontend learning stays easier to track:

- UI structure
- React concepts
- component architecture
- state management basics
- API consumption from the frontend

## Directory Structure

Each class session follows the standardized layout:

```text
class-name/
|
+-- class-notes/    -> handwritten or markdown notes from class
+-- class-code/     -> instructor reference code or cohort code
+-- practice/       -> my personal code practices for the topic
+-- assignments/    -> official cohort assignments or self-added follow-up tasks
+-- README.md       -> high level summary of the class
+-- links.md        -> resource links and tracking status
```
